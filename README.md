# Myntra Frontend Clone

A responsive Myntra frontend clone built using HTML, CSS, and JavaScript.

## Features
- Responsive Homepage
- Product Listing Section
- Add to Bag Functionality
- Dynamic Cart Count
- Clean UI Design
- Separate Bag Page

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Project Structure
- HTML for structure
- CSS for styling
- JavaScript for functionality
